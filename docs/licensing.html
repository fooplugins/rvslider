Responsive Video Slider licenses

There are four kinds of licenses: Open source, Commercial - Developer, Commercial - Team and Commercial - Organization

Open source license

The open source license is designed for you to use the plugin to build open source and personal projects. The Responsive Video Slider open source license is GPLv3. The GPLv3 has many terms, but the most important is how it is sticky when you distribute your work publicly. From the GPL FAQ:

If you release the modified version to the public in some way, the GPL requires you to make the modified source code available to the program's users, under the GPL.

Releasing your project that uses the plugin under the GPLv3, in turn, requires your project to be licensed under the GPLv3. If you are okay with this, feel free to use Isotope under the GPLv3, without purchasing a commercial license.

Commercial Licenses

The commercial licenses are designed so you can use this plugin in commercial products and applications, without the provisions of the GPLv3. With the commercial license, your code is kept proprietary, to yourself.
